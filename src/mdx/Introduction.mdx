# AtomGridTable Documentation

AtomGridTable (AGT) is a flexible and feature-rich table component for React applications, designed to handle complex data displays with minimal configuration.

## Version Information

This documentation is relevant for **@sanbira/atom-grid-table v1.0.14**.

## Installation

```bash
npm install @sanbira/atom-grid-table
# or
yarn add @sanbira/atom-grid-table
```

## Overview

This documentation provides comprehensive guides and examples for using AtomGridTable in your React projects.

## Features

- **Core Table Functionality**: Display tabular data with customizable columns and rows
- **Data Management**: Built-in selection, sorting, and pagination capabilities
- **Responsive Design**: Resizable columns and adaptive layout for various screen sizes
- **Customization**: Extensive styling options and theming support
- **Developer Experience**: Full TypeScript support with comprehensive type definitions
- **Framework Integration**: Works with Material-UI and other component libraries
- **Performance**: Optimized rendering for large datasets
