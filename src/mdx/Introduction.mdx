# AtomGridTable Documentation

AtomGridTable (AGT) is a flexible and feature-rich table component for React applications.

## Overview

This documentation provides comprehensive guides and examples for using AtomGridTable in your React projects.

## Features

- **Core Table Functionality**: Display tabular data with customizable columns and rows
- **Selection**: Enable row selection with various selection modes
- **Sorting**: Sort table data by any column
- **Pagination**: Paginate large datasets
- **Resize**: Resizable columns for better user experience
- **Custom Styling**: Extensive styling options through themes and style props
- **Context API**: Customize component behavior and appearance globally
- **Responsive**: Works on various screen sizes
