# TypeScript Types Reference

AtomGridTable comes with comprehensive TypeScript definitions to enhance your development experience.

## Core Types

### TableProps

The main props interface for the AtomGridTable component:

```typescript
interface TableProps extends TableHandlers {
  colOptions: ColOption[];         // Column definitions
  rows: TableRow[];                // Table data rows
  className?: string;              // Additional CSS class
  isLoading?: boolean;             // Shows loading state when true
  selectedRows?: (number | string)[]; // IDs of selected rows
  isHasSelect?: boolean;           // Enable row selection
  tableTheme?: string;             // Custom theme class name
  selectionArea?: string;          // Grid area for selection highlights
  setSelected?: (selected: (number | string)[]) => void; // Selection change handler
  paginationOptions?: PaginationOptions; // Pagination configuration
  sortingOptions?: Partial<SortingOptions>; // Sorting configuration
  tableStyleOptions?: TableStyleOptions; // Style configuration
  isPagination?: boolean;          // Enable pagination UI
}
```

### ColOption

Defines the configuration for a table column:

```typescript
interface ColOption {
  label?: string;
  tooltip?: string;
  name?: string;
  width?: string;
  resizeOptions?: {
    min: number;
    max: number;
  };
  isResizable?: boolean;
  isHeadersColumn?: boolean;
}
```

### TableRow

Defines a row in the table:

```typescript
interface TableRow extends Omit<React.HTMLAttributes<HTMLDivElement>, "content"> {
  cells: (TableCell | undefined)[];
  isActive?: boolean;
  selectIdentifier?: number | string;
  isHeader?: boolean;
  isSticky?: boolean;
}
```

### TableCell

Defines a cell in the table:

```typescript
interface TableCell extends Omit<React.HTMLAttributes<HTMLDivElement>, "content"> {
  content: ReactNode;
  span?: number;
  isOneLine?: boolean;
  isNumber?: boolean;
  isHeader?: boolean;
  isNoPadding?: boolean;
  isDisabled?: boolean;
  isCentered?: boolean;
}
```

## Style Types

### TableStyleOptions

Controls the visual appearance of the table:

```typescript
interface TableStyleOptions {
  isFirstRowHeader?: boolean;
  isZebra?: boolean;
  isNoXCellBorders?: boolean;
  isSmallCellPadding?: boolean;
  isStickyHeader?: boolean;
  loaderRowsCount?: number;
  colorScheme?: "light" | "dark";
}
```

## Pagination Types

### PaginationOptions

Configure pagination behavior:

```typescript
interface PaginationOptions {
  rowCount?: number;               // Total number of rows for pagination
  pageSizeOptions?: number[];      // Available page size options (e.g. [10, 25, 50])
  pageSizeKey?: string;            // Key for storing page size preference
  getApiParams?: (page: number, pageSize: number) => unknown; // Convert to API params
}
```

## Sorting Types

### SortingOptions

Configure sorting behavior:

```typescript
interface SortingOptions {
  defaultOrdering?: string;        // Initial column to sort by
  defaultDirection?: SortingDirection | null; // Initial sort direction
  resetPage: () => void;           // Function to reset pagination on sort change
  getApiParams?: (ordering: string, direction: SortingDirection | null) => unknown; // Convert to API params
}

enum SortingDirection {
  ASC = "asc",
  DESC = "desc"
}
```
